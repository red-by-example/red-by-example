The word $$add$$ adds 2 values together.
This is equivalent to the infix operator $$+$$.

Has 2 operands, which can be expressions or values
.pre
.box Add some $$number!$$s

add -1 3
== 2

add 4 / 2 5
== 7
%%4 / 2 == 2%%
%%2 + 5 == 7%%
%%Returns 7%%

add 2.3 7.77 - 2
== 8.07
%%7.77 - 2 == 5,77%%
%%2.3 + 5.77 == 8.07%%
%%Returns 8.07%%

.box Add a $$number!$$ to a $$pair!$$ and vv

red>> add 2x4 5
== 7x9

red>> add 5 2x4
== 7x9
