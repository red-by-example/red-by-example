The word $$true$$ has the logical value "true" 
and its $$datatype!$$ is $$logic!$$

Logical values are produced from various comparisons.

The $$logic!$$ type also provides:  
    $$false$$, $$no$$, $$off$$, $$on$$, $$true$$, $$yes$$
.pre
3 = 3.0
== true

3 == 3.0
== false

3 = 4
== false

"abc" = "abc"
== true

"ABC" = "abc"
== true

if 4 > 3 [print "it is"]
-- it is

a: true
== true

b: 4 > 3
== true
